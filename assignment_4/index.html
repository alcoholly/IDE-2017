Description

The dataset consists of 40 data points with 112 dimensions each. The dimensions represent the contours of hands of different shapes. To plot this in a scatterplot, a script performing PCA on the dataset is provided, where we plot the first two (the largest) dimensions in the scatterplot.
The make it easier to use the PCA data, we modified the python script to output only the two first values along with x and y as the labels.
The large image of a hand can be updated by either clicking one of the small hands above it or by clicking one of the circles in the PCA plot.

Results and conclusions

When looking at the data in the scatterplot and visualising it, it seems that a lower x value (the first domension in the PCA data) corresponds to more separated fingers overall, or perhaps the distance between the two outmost fingers. The y value seems to correspond more to the distance between the little finger and the other fingers.

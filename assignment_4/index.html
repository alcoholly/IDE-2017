Description

Above we present a 2-panel visualization based on a dataset of hands of different shapes and finger extensions (hand set) that consists of 40 data points with 112 dimensions each. These dimensions represent the x and y coordinates of the (56) points that form the contours of the hands in the dataset. All the hands outlines are presented on the top left panel of the visualization and correspond to the geometric representation of the data. At the bottom of the left panel, a larger version of one (user-selected) hand outline is presented. The right panel of the visualization shows a scatter plot that is produced by projecting the data points onto the first 2 principal components of the data. These values capture the largest variance in the data and were obtained by using a provided python script that performs principal component analysis (PCA) on the hand set. PCA is a statistical procedure that allows for dimensionality reduction in high dimensional data sets while preserving the important information in the data. The first principal components explain most of the variation in the data and thus, PCA allows us to summarize and and to visualize the information of high-dimensional data based on only few variables /dimensions. For the purposes of this project we present 2D visualizations of the data, hence the python script was modified to output only the values of the first two principal components along with x and y as the labels.

Methods

When the user either clicks on one of the small version of the hands outlines (top-left panel) or on one of the points in the PCA plot (right panel), then the larger version of the outline (bottom-left panel) is updated and the corresponding hand outline is shown. Moreover, by clicking on one of the small hands-outlines the corresponding point in the scatterplot is highlighted so that the user can easily map between the hand outlines and their corresponding PCA projections. Lastly, the user has the option to hover over a point in the scatterplot and have a small box display its respective PCA values.

Results and Conclusions

The 2-panel visualisation and the mapping between the 2 types of representation allows us to directly compare the geometric representations of the hands to their lower dimensional analogues. We can observe that the datapoints of similar postures are clustered closely together while distant datapoints suggest that their respective outlines are quite different. If we try to interpret what type of differences each component captures we can notice that a lower x value -the first PCA dimension that accounts for largest possible variance in the data- generally corresponds to more separated fingers. Taking into account the bottom-left datapoint that has the second-lowest x value (~ -0.42) while the index, the ring finger and the small finger are positioned closely to each other, we could rephrase this observation by saying that lower x values are associated with higher distance between the two outermost fingers. The y value (second principal component) on the other hand, seems to correspond more to the distance within the fingers- although its role in the underlying differences between the hand outlines is not as clear as in the case of the first principal component. Overall our dataset is quite small and the differences between the datapoints are not very large. If we worked with a different dataset that would contain more datapoints with greater variance (eg a hand set that contained hand gestures of sign language) we would expect the projection onto the 2 first principal components would form more distinctive clusters.

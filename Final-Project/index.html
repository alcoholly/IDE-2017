Last.fm data visualisation

Introduction

For this final project we have made use of datasets from the last.fm API. One of the datasets consists of 601 artists mapped to similar artists, where the size of the node is determined by its degree (number of adjacent nodes). The other dataset consists of music "communities", which are determined by tags from the artists. The data was collected as JSON, and for our second visualisation we used Python to convert this to csv, which made it easier to load the data in the library we were using.

Visualisation 1

The graph in our first visualisation shows the relations between artists. The node sizes are determined by their degrees.
The search box highlights any matches for the search string. Zoom can be achieved by scrolling.

Visualisation 2

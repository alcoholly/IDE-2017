Motivation, method, conclusions

The data we chose is showing temperatures in St. Petersburg from 1880-2017.
Since the raw data isn't proper csv (the entries are separated by multiple spaces, not comma or tab), we normalized it with sed:

sed 's/ */,/g' < station.txt > station_normalized.csv

The first chart is the one we created from scratch (although in order to learn how to do it we did look on different resources like the lecture notes and the Let's make a bar chart site so there are probably some similarities). Since we wanted to make a bar chart, and since it would just be too crowded using every year, we wrote a Python script to get the mean temperature for each decade (based on the mean temperature for each year) instead, since it still shows that the mean temperature is increasing. Since the current decade is still "ongoing", we chose to leave it out for the decade data. It also allowed us to work with different datasets of data so that we don't have to present the exact same data twice.
